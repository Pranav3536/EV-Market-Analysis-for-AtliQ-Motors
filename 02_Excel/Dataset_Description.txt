Dataset Description

This project uses two Excel datasets:

1. AtliQ_EV_Raw_Dataset.xlsx
   - Contains the original EV market data collected for analysis.
   - Includes raw records with missing values, duplicate entries and inconsistent data that required cleaning before analysis.

2. AtliQ_EV_Clean_Dataset.xlsx
   - Final cleaned dataset prepared after data preprocessing.
   - Used for SQL analysis, Power BI dashboard development and business insights.

Dataset Features

The dataset contains information related to the Indian Electric Vehicle (EV) market, including:

Column Description
- Date –-  Date on which the sales data was recorded.
- Fiscal Year –-  Financial year corresponding to each record.
- Quarter –-  Fiscal quarter (Q1, Q2, Q3, Q4) of the transaction.
- State –-  Indian state where the vehicle sales were reported.
- Vehicle Category –-  Category of the vehicle (2-Wheelers or 4-Wheelers).
- Maker –-  Name of the EV manufacturer.
- Electric Vehicles Sold by Maker –-  Number of EV units sold by each manufacturer.
- Electric Vehicles Sold by State –-  Total EV units sold in each state.
- Total Vehicles Sold –-  Overall vehicle sales (EV + Non-EV) in the state.
- EV Penetration (%) –-  Percentage of EV sales compared to total vehicle sales.
- EV Penetration Category –-  Classification of states based on their EV penetration level.

The cleaned dataset was validated to ensure data consistency and improve the accuracy of the analysis.
